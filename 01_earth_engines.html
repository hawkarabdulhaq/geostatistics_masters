Option A ‚Äî Quickest (Code Editor path)

4) Your very first run (in the Code Editor)

  2. Paste this JavaScript and click Run:
JavaScript ‚Äî Landsat 8 over Budapest (true color) 
// Load a Landsat 8 image over Budapest and display true color
var img = ee.ImageCollection('LANDSAT/LC08/C02/T1_L2')
  .filterBounds(ee.Geometry.Point(19.0402, 47.4979))
  .filterDate('2023-06-01', '2023-08-31')
  .sort('CLOUD_COVER')
  .first()
  .resample('bicubic');

var rgb = img.select(['SR_B4','SR_B3','SR_B2']).multiply(0.0000275).add(-0.2);
Map.centerObject(rgb, 10);
Map.addLayer(rgb, {min:0.03, max:0.3}, 'L8 True Color');

(Any beginner quickstart/tutorial in the docs will also work the same way.) [Google for Developers]

5) (Optional) Set up Python

  1. Authenticate & initialize in a notebook or local Python:
